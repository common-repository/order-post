Enables you to select particular category and arrange the posts of that category in a order of your choice by simply drag and drop. 

== Description ==
Order Post plugin is developed for ordering the posts of a particular category in a particular sequence as we need by simply drag and drop the 
posts of that category . This plugin provides the shortcodes for using this plugin . you can use this plugin either in Pages , Posts , Widgets or even in comment section to use this plugin. Wherever you use as per your requirement 
this plugin will output the ordered posts in that area.
**Language Support:** The plugin interface in only in English.


= Some of the features added by this plugin =

* You can select the category and order the posts of that category that you have selected as per your wish.
* Fully ajax is used for providing drag and drop facility.
* User friendly.
* Full explanation is given in plugin itself . So, even a new user caan also use this plugin.



== Installation ==

a) Best is to install directly from WordPress. 

b) Manual installation 


1. Put the plug-in folder into [wordpress_dir]/wp-content/plugins/
2. Go into the WordPress admin interface and activate the plugin
3. Optional: go to the options page and configure the plugin


If you go for manual installation -"please make sure all of the plugin files are in a folder named "order posts" in the plugin directory".


== How To Use ==

How to select the category
==========================
Welcome to the post arranger

Below you can see the list of all the categories You can select any category and just press the save button to add the category you want to order the posts .

Note: the category that is selected and saved, the

corresponding posts of that category will be visible in posts arranger option where you have to simply drag the posts title and the order will be saved automatically if you go to the posts arranger option you will get the shortcode just place this shortcode in place of your normal query posts that you will write to populate your posts of this category and your posts will be ordered according to your set order using this plugin.

Hope the above explanation is clear and make sense.

Here you will get all the shortcodes for the categories that you have added through this plugin just paste this Shortcodes either in Posts,Templates,Widget,Comments and you will get all the posts ordered according to the order that you have set in Post Arranger option

How to use shortcode
=====================

1. Inside the php page : <?php echo do_shortcode('[vs-posts-order vs_limit="limit_number"  vs_cat="category_number_generated"]') ;?>

2. Inside the posts : [vs-posts-order vs_limit="limit_number" vs_cat="category_no_generated"]

Note*: 1. You can set the limit of the posts by entering the value corresponding to the vs_limit, that how many posts that you want to display. Default limit is 1.

2. Do not remove the value corresponding to the parameter vs_cat If you will change or remove, the plugin will either not work or display the posts of Uncategorized


How to Arrange the Posts
========================
 On RIGHT-SIDE you can see the category that is visible. Just press the show posts button to see the posts of that category for which you want to order the posts .

Note: the category that is selected and saved, the

corresponding posts of that category will be visible in posts arranger option where you have to simply drag the posts title and the order will be saved automatically.

Hope the above explanation is clear and make sense.
